**CLIRuntime class with these methods:**

1. `__init__(self, config: AgentConfig)` — stores config, initializes `_state = CliState.BOOTING`, `_running = True`, `_child: pexpect.spawn | None = None`, `_redis: aioredis.Redis | None = None`, `_http: httpx.AsyncClient | None = None`, `_executor = ThreadPoolExecutor(max_workers=1)`.

2. `async def run(self) -> None` — Main entry point. Full boot sequence per D-08:
   - Publish BOOTING state
   - Initialize Redis connection (`aioredis.from_url` using `redis://redis:6379`)
   - Initialize httpx.AsyncClient
   - Call `_credential_gate()` — checks credentials, HITL if missing
   - Publish READY state
   - Register with Registry via `POST {gateway_url}/registry/agents` with `{"agent_id": ..., "capabilities": ..., "boundary": ...}`
   - Enter `_task_loop()`
   - On exit: deregister, close connections

3. `def stop(self) -> None` — Called by SIGTERM signal handler. Sets `self._running = False`. If `self._child` is alive, schedules `_graceful_shutdown()` as asyncio task.

4. `async def _credential_gate(self) -> None` — Checks `_credentials_present(self.config.runtime)`. If missing: publish CREDENTIAL_WAIT, send HITL `request_user_input` via Gateway POST /actions with message asking user to `docker exec -it <container> claude auth login`, then call `_wait_for_credentials()`. If credentials appear, continue. If timeout (1 hour), log error and exit.

5. `def _credentials_present(self, runtime: str) -> bool` — Check `CREDENTIAL_PATHS.get(runtime)` exists and `stat().st_size > 0`. Return False for unknown runtime.

6. `async def _wait_for_credentials(self, runtime: str, timeout_s: float = 3600.0) -> bool` — Use `watchfiles.awatch()` on credential directory. If `awatch` is None (not installed), fall back to polling loop with 5s interval. Return True when credentials appear, False on timeout.

7. `async def _task_loop(self) -> None` — While `self._running`: poll Broker `GET {broker_url}/tasks/next?capabilities={caps}`. On 204: sleep 2s and retry. On 200 with task JSON: publish BUSY, call `_execute_task(task)`, publish READY (unless state is CREDENTIAL_WAIT from auth failure).

8. `async def _execute_task(self, task: dict[str, Any]) -> None` — Extract task_id and message. Call `_run_cli_process(message)` to get `(exit_code, stdout)`. If exit_code == 0: post result as success via `POST {gateway_url}/tasks/{task_id}/result` with `{"task_id": ..., "agent_id": ..., "result": {"status": "completed", "output": stdout}}`. If non-zero: classify failure. If auth_expired: transition to CREDENTIAL_WAIT, send task_failed, call `_credential_gate()` (D-16 no retry). If other failure and first attempt: retry once (D-15). If still fails: send task_failed with typed reason via `POST {gateway_url}/tasks/{task_id}/result` with `{"task_id": ..., "agent_id": ..., "result": {"status": "failed", "reason": reason, "output": stdout}}`.

9. `async def _run_cli_process(self, task_message: str) -> tuple[int, str]` — Build command via `_build_command()`. Spawn via `pexpect.spawn(cmd_str, timeout=None, encoding="utf-8", dimensions=(10000, 200))`. Store in `self._child`. Run `_drain_to_buffer` in thread pool executor. On EOF/exit: `child.close()`, get `child.exitstatus or 0`, clear `self._child`, return `(exit_code, output)`.

10. `def _build_command(self, task_message: str) -> list[str]` — Build command list:
    ```python
    cmd = ["claude", "-p", task_message, "--output-format", "json",
           "--dangerously-skip-permissions", "--no-session-persistence"]
    if self.config.model:
        cmd += ["--model", self.config.model]
    return cmd
    ```

11. `def _drain_to_buffer(self, child: Any, buf: list[str], task_id: str) -> None` — Blocking drain in thread pool. Uses `child.read_nonblocking(size=4096, timeout=0.1)` in a loop. On TIMEOUT: check `child.isalive()`, break if dead. On EOF: break. Appends chunks to `buf`. Tracks total bytes and truncates at MAX_OUTPUT_BYTES with warning. Also accumulates progress chunks and batches them (but progress posting is async — use a simple list accumulator and let the caller handle async posting after drain completes).

12. `def _classify_failure(self, exit_code: int, output: str) -> str` — If exit_code == 0: return "". Scan last 50 lines lowercased against FAILURE_PATTERNS. Return first matching reason key, or "cli_crash" as default.

13. `async def _publish_state(self, state: CliState) -> None` — Publish JSON `{"agent_id": ..., "state": state.value, "timestamp": datetime.utcnow().isoformat()}` to Redis channel `lifecycle:{self.config.agent_id}`. Wrap in try/except — state publish must never block.

14. `async def _post_progress(self, task_id: str, content: str, *, final: bool = False, sequence: int = 0) -> None` — POST to `{gateway_url}/tasks/{task_id}/progress` with ProgressUpdate-shaped payload. Wrap in try/except.

15. `async def _request_hitl(self, message: str) -> None` — POST to `{gateway_url}/actions` with `{"action": "request_user_input", "agent_id": self.config.agent_id, "payload": {"message": message}}`. Wrap in try/except.

16. `async def _graceful_shutdown(self) -> None` — If `self._child` is alive: `child.terminate(force=False)` (SIGTERM). Wait up to 5s in 0.1s increments checking `child.isalive()`. If still alive: `child.terminate(force=True)` (SIGKILL). All wrapped in try/except.

**Important implementation details:**
- Use `ThreadPoolExecutor(max_workers=1)` for the pexpect drain loop to avoid thread-safety issues (research open question 1)
- Spawn Claude Code with `cwd="/app"` in pexpect to ensure CLAUDE.md is picked up (research open question 3)
- `_drain_to_buffer` runs in `loop.run_in_executor(self._executor, ...)` to avoid blocking asyncio
- All HTTP calls use the shared `self._http` AsyncClient
- All Redis calls use the shared `self._redis` connection
- Pre-flight credential check before each task (Pitfall 3): call `_credentials_present()` at the start of `_execute_task`, transition to CREDENTIAL_WAIT if expired
